Git clone

```powershell
git clone https://github.com/phykn/phykn.github.io.git
```

Git push

```powershell
git add .
git commit -m "some text"
git push
```

Execute Serve

```powershell
bundle exec jekyll serve
```

Delete Directory

```powershell
rmdir /s /q phykn.github.io
```
